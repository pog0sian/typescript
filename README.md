# TypeScript Utilities

Кратко: набор небольших утилит и примеров на TypeScript (immutable операции над массивами, in-memory репозиторий, рекурсивный `DeepReadonly`, пайплайны функций, разбиение массива на чанки).

## Требования
- Node.js (рекомендуется >= 14)
- npm
- Для разработки можно использовать WebStorm (проект разрабатывался на macOS)

## Установка
```bash
npm install
